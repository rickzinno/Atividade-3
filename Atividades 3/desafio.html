1. Condição Simples (if)

📌 OBS: A condição "if" executa um bloco apenas se a condição for verdadeira. Aqui verificamos se a idade é maior ou igual a 18.

2. if / else

📌 OBS: "if/else" cria dois caminhos: se a condição for verdadeira, executa um bloco; se for falsa, executa o outro.

3. Condição com E (&&)

📌 OBS: O operador "&&" (E) exige que TODAS as condições sejam verdadeiras para executar o bloco.

4. switch / case

📌 OBS: "switch/case" é útil para múltiplas opções. Compara o valor com cada "case" e executa o correspondente.
